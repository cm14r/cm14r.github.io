亲吻阴蒂—2

…
我又一次舔舐着她的阴部。
舌部感受到，铁锈般甜腥的血液与咸味的体液混杂，经由舌尖进入我的口腔中。
如同丧失理智一般，我疯狂渴求着她的血与体液。
“唔…嗯…”
舌部运动得越来越快。
想要让她高潮。
又有血流了出来。
我吸食着她的血液与流出来的体液。
…
癫狂。
我如同疯子一样渴求着她。
她的经血与体液，她高潮的模样，她。
全部都属于我。
“哈——啊————哈啊————唔——！！”
她的头部向后仰去，身体止不住一阵阵地拱起，颤抖。
她高潮了。
又一次，被我舔舐到高潮。
叮———铃———铃
突然，一阵手机铃声响起。
不是我的手机铃声…
“晨，我接下电话。”她对我说。
“嗯，好。”
她拿起了手机。
我能感觉到她的神情不对劲。
“抱歉，我要出去一下。”
“额…？”
她在身上披了个毯子，然后走了出去。
“…”
怎么回事？
…
…
“你有什么事？
“亲爱的妹妹大人，你跑去哪了？
“我没有义务告诉你。
“你就这么对哥哥说话？”
“到底有什么事？”
“只不过回来了，想你了而已。
“恶心。
“你确定你要这么对哥哥说话？
“你知道吗？现在我就想把你剁成烂肉，然后油炸成肉饼。
“妹妹…？你在说什么呢…？对哥哥这么残暴吗？”
“这是我真实的想法。”
“你不爱我了吗？我可是你哥啊。”
“别意淫着乱伦了。”
“什…么？你在说什么？我…才…没有想这些！我是真的爱你！”
“我没有义务和你说话。”
我挂了电话。
啧。
…
…
“抱歉，久等了。”
“没事的，也没多久…”
“那个…今晚…还继续吗？”
实际上，我没什么兴致了。
但我能感觉到，她似乎在期待着。
…
“嗯。”
…
…
她的舌头舔舐着我的阴蒂。
不久前，她连自慰都完全不会，她从来都没有这么做过。
甚至不知道，女人靠阴蒂高潮。
这个社会，女人自慰似乎是禁忌中的禁忌，绝不可触碰的事物。
男人将女性自慰污蔑为肮脏污秽的行为。
而男人真正肮脏污秽的行为却无人批判。
…
她似乎更快地在舔舐了。
我知道，她在期待着我的高潮。
但我现在却难以做到。
没有感觉。
要假装高潮吗？
…如果有必要的话。
一直不高潮的话，她可能会很累而且伤心的。
…就这么办吧。
…
…
她在我的身边入眠。
在我“高潮”之后，我要求用手指让她高潮。
在她的身后，我先是用手指轻抚着她的阴蒂，然后越来越快。
她的身体止不住地震颤。
我让她高潮了三次。
她的每次反应都十分强烈。
她的欲望，或许真的很强吧。
在第一次高潮，她不由自主地扭头的时候，面对她的脸庞，我情不自禁地将嘴唇重叠上她的嘴唇，同时抚摸着她的阴蒂。
这样做了之后，我感到她抵达高潮的速度似乎变快了，震颤的幅度也变大了。
但是，这样做果然还是不正确的。
尊重，而非自作主张。
之后，我对没有询问她就如此做而感到抱歉，但她似乎不介意，也叫我完全不用在意。
仔细想想，之前我已经有很多次没有尊重她了。
偶尔，在她面前我会失去理智，变得任性。
为什么会这样？
…
啊。
已经是凌晨了…今天还要工作啊。
这样会睡不着的。
嗡————
我的手机振动。
？
我打开手机。
…是那个人发的消息。
换手机号了啊。
“妹妹，你在****小区干什么呢？我到楼下了，来接我吧。”
…
来吧。
…
…
“哎呀，妹妹大人，见到你了哦，在这里做什么呢？跟哥哥一起回家吧。”
“…”
“妹妹，你可能对我有点误会，我们回家聊，好吗？”
“…”
我逐渐接近这个男人。
“妹妹…？
然后——我一脚踢向了男人的下体————并用厚毛巾将男人的嘴捂住。
男人双目瞪大，看向我。
“如果你敢报警，我就把你的那些龌龊事全部暴露。”我威胁男人。
“妹妹…”
“不要再骚扰我。”
“弟弟的死亡是因为你吧？”
“但是…那是你…你叫我…是你叫我这么做的…”
“我当时才几岁，教唆你？你在开什么玩笑？你有证据吗？”
“我…我…没有。”
“我在录音。”
“哎？？”
“从你骚扰我到你亲自承认害死弟弟，全都录下来了。”
“不…不…妹妹…求你别…”
“我给你当狗好吗？就像以前那样！别把这些事告诉别人…”
“别再骚扰我。”
“妹妹…”
男人在原地抱头痛哭。
我得走了。
对于这个蠢男人，没有什么可说的。
我现在期望的是，这个男人被绞肉机绞烂。
…
…
打开门，她睡眼惺忪地站着等我。
“唔…你去哪了…？”
“对不起，没什么。”
“继续睡觉吧…今天白天我课排满了…”
“嗯。”
继续睡觉吧。
…
…
又是…电话？
“是吗…我知道了。”
“发生这种事我感到很难过，我会抽空处理的。”
我挂了电话。
…
早上————
“唔…唔…”
“醒了？”
“嗯…”
“抱一个吧？我要走了。”
“诶？不是才七点左右吗…”
“有点事要去处理。”
“那…下次什么时候能见到？”
到时候，校门外等我吗？
“到时候，我打电话给你！”
和大部分时候不一样的回复…
“怎么了？”
“没什么…”
“别太得寸进尺了，我。”
“是吗？那么——“
“抱一个吧————“
“她抱住了我。”
“两人的肌肤紧贴。”
…
…
中午回来休息。
自己用手自慰…
没有你…好痛苦…
好压抑…
明明才分离了没多久…
依恋你，渴望你。
无法高潮。
好痛苦。
眼泪几乎要流出来。
想要触碰你。
可以给你发消息吗？
…
嗡嗡————
手机响了，是你的消息吗？
我迫不及待地拿起。
是你的消息。
但是…
“最近两三天可能都不能见面了，抱歉。”
你如此说。
心脏好痛。
要无法呼吸了。
我到底是怎么了？
明明只是分别了一会儿而已…
最终，我也只能打出“嗯，好的。”这几个字回复你。
好想对你任性。
但是，不可以。
那样，会给你添麻烦的，你可能还会讨厌我。
嗡嗡————
手机又响了，会是你的消息吗？
“晚上的时候，我会给你发视频通话的，要接哦！”
对了…
还可以这样…
这样就可以看见你的面庞了。
“话说…为什么你几天都不能和我见面…？”
我打出这几个字。
但又删除掉。
别太得寸进尺了。
真当自己是谁了。
“嗯嗯，我知道了！”我打下这几个字，点击了发送。
不要任性。
睡午觉吧。
还有课呢。
