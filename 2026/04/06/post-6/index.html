亲吻阴蒂——6

我拿起铲子，刨坟。
一下，两下，三下…
铲铲铲！！
我的身上，全是泥土。
“你在做什么呢？”
突然，一道声音传到我的耳边。
“哇啊啊啊啊啊啊！！！！”
我转过头，那是一个姊姊。
穿着与这里环境格格不入的西装…
看起来可能比我大十岁左右…？
“抱歉，吓到你了。”
“诶？姊姊你不是来阻止我的吗？”
“为什么要阻止呢？这些东西和我没关系啊。”
“话说…姊姊，我没见过你呢…”
“我是隔壁村的。”
“额…是吗？”
“就是之前祠堂被烧毁的那个村～”
“噢！我知道了！！”
“所以，你是在刨坟吗？”
“嗯…”
“那么，加油吧！”
“…？”
加油？
“要来两颗糖果吗？”
“我要——！但是…陌生人的东西不能随便吃…”
“是进口巧克力哦。”
“进口？巧克力？什么东西啊？”
“尝试一下吧？”
她的手上是精美包装的食物。
看起来不会有蠹的样子。
从她身上也感受不到像村人一样的恶意…
试一试吧…
我接了过来。
打开包纸。
然后——含入口中。
甜蜜与苦味中和的感觉在口中扩散。
不错…
“怎么样？”她问我。
“挺好吃的…”
“以后不要再吃陌生人的东西了哦。”
“诶？”
“但是，我们是朋友，所以没关系的。”
朋友？
“对吧？”
姊姊笑着看着我。
我跟她对视…
唔————
她的眼神…
我不自觉扭过头去。
“怎么了？”
“没…怎么…”
奇怪的感觉…
感觉心跳加速了…
“…不好意思，我该走了。”
诶？
这么突然吗？
“没关系的啦，或许以后还会再见呢？”她好像看出了我的情绪，如是说道。
真的吗…？
不想要她走…
明明才见面没多久…
为什么会这样呢？
但是…也没办法。
“嗯…”
“那么，给你个东西吧！”
“什么？”
“钻石哦！”
“钻石？是什么？”
“我给你看看吧！”
她拿出一个小盒子。
然后打开————
我看见，里面放了个亮闪闪的透明石头。
“这个是拿来干什么的啊？”
“卖钱。”
“？”
这个东西价值很高吗？
“总之，请你收下它吧，这里还有它的GA证书。”
她又拿出一张纸。
“证书？有什么用？”
“更好卖钱。”
“诶？”
“你可以卖掉它。”
卖掉？
“诶，卖掉它？姊姊你不会不高兴吗？”
“没事的。”
“它的价值也只有这样。”
“才不是！”
“既然姊姊和我是朋友。那么，现在的它，可以作为姊姊和我的友情信物！”
“…”
她沉默了一会儿。
“也是呢。”
“但是，如果你个人真的需要它的价值的话，那就卖掉它吧。”
“不会的哦！”
“…”
她没有说话。
…
…
我之后也有去铲坟，希望能再遇见她。
但自那之后，我再也没见过她。
她给我的钻石，我从来都不会想着去变卖。
这件事我也从来没有跟人说过。
偶尔铲坟的时候，我还能挖出骨头，比如头骨、腿骨，别的什么部位的断骨。
我会把那些骨头拿去煮汤，然后把汤水加到菜里。
我还会去偷别人家的骨灰盒和坛子。
吃那种全村人的大锅饭的时候，在饭菜撒点搅和。
看到蠢货们在吃牠们“亲人”，“爱人”的尸骨粉时，我就会感觉一种狂喜感冲上心头。
太有趣了。
太有意思了。
还有新鲜下葬的尸体。
那时候没有禁止土葬。
我在夜间偷偷跑出去，然后挖尸体。
一下，两下，三下…
尸体的面目浮现在我的眼前。
割下一块腿肉，割下一块手臂肉，割下一块脸皮肉，割下一块极为肮脏恶心的肉————一下一下，将尸体如此分割至面目全非，偶尔拆出几根骨头。
当然，我对人肉没有太多兴趣，吃得不多。
但我喜爱做这样“亵渎”之事。
太有趣了，太有趣了。
鬼在哪？
佛在哪？
诅咒在哪？
有本事就来找我吧！
我不会将坟填上。
因为我喜爱视见村人看到这幕的神情。
村人看见坟被挖，村人看见面目全非的尸体。
似是悲痛，似要呕吐，似要晕厥，似渴要将做出这件事的人杀死。
太有趣了。
太有趣了。
哈哈哈哈哈哈哈哈哈哈哈哈！！！！！！！！！
将人推下河，于不远处看见人从挣扎到完全溺水的模样。
尸体被人发现时，已经泡烂。
于夜中将几根点燃的烟丢入别人家里，在大门上面卡住东西。
火焰燃烧，消防队过了很久才来。
四死一伤。
一个家庭被毁了，真是令人痛苦流涕，哈哈哈哈哈！！！！
在半夜跟踪一个人，然后趁其不备插上几刀。
尸体一下滑落在地上。
偷来的黑色雨衣被溅上血，似铁锈味般的血腥味涌入鼻腔，令人感到愉悦。
这一切，太有趣了，太有趣了，太有趣了哈哈哈哈哈哈哈哈哈哈！！！！！！！
…
…
醒来。
嘴唇，柔软的触感传来。
喜欢。
但…又要分开了…
…
…
今天，我的学校放假了。
但是恒晖还要去公司。
真辛苦…
但没办法…
对了，那个钻石…
我记得，是放在————
…
…
我打开盒子。
还是如以前一样，闪耀的钻石。
尽管这样的它，作为装饰品的它毫无用途。
现在钻石似乎也跌了不少了。
…
反正怎么样也不会卖掉它。
…
对了。
…她…有在好好地活着吗？
当时的你，又为什么要将这么贵重的东西给我呢？
完全不理解…
我连她的脸都记不清…
算了。
别闲着想这些了。
送外卖去吧。
忙起来就不会想这些了。
我将钻石盒揣入兜中。
嗡嗡——
新消息…
我打开手机。
“不要趁我不在去兼职哦。”
…
好凑巧。
嗡嗡——
她转给了我3000元。
不可能收下啊…
“好。”
我无视转账，说了谎。
还是送外卖去吧。
…
…
中午————
我吃完饭，走出了餐馆门。
突然，有什么人好像在扯我的衣角。
“？”
我望向下方。
是一个女孩。
“有什么事吗？”
“我…跟妈妈们走丢了…能拜托你打个电话给妈妈们吗？”
“好。”
我不假思索地回答了。
“一个妈妈的电话是…1……*”
“打过去了，你接吧。”我将电话递向她。
“嗯嗯。”她接过了电话。
…
…
“好吃，好吃！！”
打了电话后，因为听到她肚子饿了，所以带她去了餐馆吃饭。
“这个肥肉软软糯糯的，好好吃～～谢谢姊姊！”
“没什么，慢点吃。”
“姊姊就像妈妈们一样～～”
“…”
“小瑞！”两种声音重叠在了一起。
“啊，妈妈们来了！嗨！！”
…
…
“这完全是我们这边的责任，抱歉麻烦您了。”其中一位寸头的女子说着。
“我们这边补偿您两千元吧。”另一位留有短发的女子说。
“啊？”
“是少了吗？您说个数。”短发女子说。
“太多了吧…”
在这里遇见的好多人感觉都没什么金钱观念…是太富有了吗…
“我认为这不算多，感谢您没有让小瑞遇见危险。我们赔偿您的误工费。”寸头女子说。
“没有啊…我什么都没有做的…我不要钱…”
“我这里带有些现金，您收下吧。”
短发女子似乎从口袋中拿出一些钱。
“不…我真不要！！”
我拼命摆手。
但短发女子还是将钱塞进了我的口袋里。
…
…
“谢谢您啦！姊姊！！拜拜！！！”
她们上了一辆黑色的车，女孩在向我挥手道别。
“拜拜。”
我也挥手向她道别。
车一会儿就开走了。
刚才拿到的钱….
我从兜里拿了出来。
“？”
刚才没仔细看…
居然是外币…
六张50面额的…
我查了下汇率。
换算过来…差不多两千七…
这地方怎么回事…感觉遇见了好多有钱人…这钱都随便撒的吗…
这钱…要去银行换吗？
看了下…现在感觉是汇率高点了。
…
算了，留着吧。
感觉也挺稀奇的。
…
…
刚又送了两单外卖。
我下了电梯，走出大门和小区。
一个电话打了过来。
是之前那个女孩说的号码。
似乎是她其中一个母亲的电话。
打过来有什么事吗？
我接通电话。
“姊姊，我安全到家了哦！”
是那个女孩…
“那真是太好了。”
“姊姊，不开心吗？”
“没有。”
“啊—哦，姊姊不会在工作吧？”
“算也不算吧。”
现在手上没单了。
“打扰到你了？抱歉哦！”
“没有的事。”
“今天偷跑出来忘带手机了，麻烦你了，姊姊，抱歉。”
“没什么的。”
“话说，姊姊是大学生吗？”
“额…是…”
“难道在**大学读？？”
“…嗯。”
她怎么会知道的？
“居然被我猜对了！！”
“那所大学超难考的啊！！姊姊好厉害！！”
“…”
“姊姊，可以考虑当我的家教吗？”
诶？
…
…
回家时，她已经躺在了沙发上。
也没有很晚啊…她居然都回来了…
额？这么一说，难道我希望她回来得晚吗？
怎么可能…
…
…
我靠近她。
她躺在沙发上睡觉。
真要命。
受不了。
她的睡颜。
我的体温，我的心跳。
想要亲吻她。
我不自觉地想要将嘴唇靠近她的脸————
正当我的嘴唇差点亲吻上她时————
她睁开了眼睛。
“诶？”
我愣了。
“不仅违反约定去兼职，而且还想要趁我睡着的时候做这种事吗？”
“对不起啊啊啊啊！！！！！”我疯狂道歉！！！
“算了，既然阻止不了，那你就做吧，我不阻拦你了。”
“真的吗？”
“我指的是兼职。”
“正好，关于兼职，我有话跟你聊下。”
“什么？”
…
…
“家教…薪资也很高，挺好的。”
“你同意了？”
“刚才就说了，我不阻止你。”
“好诶！”
“但是，明天不能去。”
“为什么…？”
“明天我会一直待在这。”
“我明白了！！”
…
…
