亲吻阴蒂——7

…
…
我们驾车去了自助餐厅。
…
…
环顾四周，大家基本上拿的都是海鲜和牛排什么的…
虽然有小炒和饭的自助区，但没有勇气去拿…
本身价格就好贵啊…
不能吃得太没价值。
但是，我不喜欢吃那些啊…
“没关系的，什么都可以拿。”
她似乎看穿了我，微笑地说着。
“但是…几百块钱，我只吃十几二十块钱的东西…好吗？”
“吃饭吃的就是自己喜欢吃的东西。”
“如果昂贵的东西不能带给自己幸福与满足，那也没什么好的。”
“谢谢…”
“没事的，去拿东西吧，不要饿着肚子了。”
…
…
吃饱喝足。
几百块钱自助的小炒就是不一样。
真好吃。
…
…
我们一同走出了餐馆。
突然，一个电话打了过来。
“你好，你有一个快递。”
“什么？我根本没买东西。”
“但是电话就是你的，寄到大学的。”
还是寄到学校的？
是那人的“惊喜”吧？
我倒是想看见到底会给我怎样的惊喜。
“放在代收点吧。”
说罢，我挂了电话。
“现在要去取件吗？”她问我。
“嗯。”
…
…
我戴上手套和口罩，在代收点用剪刀打开了快递。
撕————
撕剪开纸箱的一瞬间，浓烈的恶臭味散发开来。
拆开来，里面是一只被分尸多段的猫。
猫眼睛睁大，头颅被剁了下来，腿脚被砍断，使得纸箱壁非常脏污。
无聊。
想用这种东西威胁我什么？
咚————
我把纸箱连同手套口罩直接丢进了店外的垃圾桶。
猫肉可不好吃，不新鲜的猫尸应该口味会更糟。
狗肉煲的味倒还不错，皮质富有弹性，似羊肉般的口味，虽也具有膻味，但无伤大雅。
好久没吃到了。
村里的狗基本都是散养的，看着就跟无主的似的。
咬人嘛，也就不认就是了，被“流浪”狗咬到抓到就自认倒楣吧。
所以，被我抓来吃狗肉也自认幸运吧。
反正一堆人闻到狗肉香味的时候不也把持不住吗？
真是香啊，真是美味啊。
夹起一块“大黄”，“旺财”的肉，咽下肚中。
与羊肉的细微差别，不会有人在意的。
夹起一块“祖父”，“儿子”的肉，咽下肚中。
与牛肉的差别，谁在乎呢？
失踪的狗，被割烂的尸体。
只要美味就好了。
只要愉悦就好了。
将几颗人头挂在祠堂的正中心，在祠堂的墙壁和雕像上写下侮辱性话语，打翻供奉台。
让村人看见牠们的“祖宗”被侮辱，供奉台被打翻。
让村人看见作为牠们“祖父”，“父亲”，“儿子”，“兄弟”等身份的尸体被人侮辱亵渎，头颅被人残忍砍下，然后悬挂在了祠堂的吊顶之上。
我在旁边看着村人的晕厥，听着村人们的诅咒。
“大逆不道”，“”，“遭天谴，遭报应”，“去死”，“下地狱”…
村人们将矛头对准一个天天抱怨政治历史的男人。
没过多久，这个男人跳进了河里。
在那男人死了两天之后，我去挖了别村人的坟场。
挖出来，尸体潮湿，上面还有不少蛆虫蜈蚣在爬着，被啃出一个个血洞。
我砍掉尸体的头手，并将头颅剖切开，露出尸体其中的大脑，就这样丢在了祠堂之中。
现场有人当场呕吐了出来。
村人们质疑是一个刚回来两天的女人所为。
之后，那个女人与村人们大吵一架就离开了。
女人被指是畏罪潜逃。
女人的“家人们”称在那之后联系不上女人，并将女人作为谈资一样到处八卦造谣。
因为厌恶听这些，所以我烧了那个女人“家人们”的房子。
那之后，没人提起那个女人了。
又过了几天，祠堂也被一场大火烧着。
人们的惊恐、痛苦、无助与愤慨。
狰狞扭曲的神情。
一切都是如此令人狂喜。
近乎要令人疯癫的愉悦感冲上心头————
…
…
“我回来了”
嘭————
我关上车门。
汽车发动。
…
…
我和她在沙发上坐着。
“你想知道我今天发生了什么吗？”
“发生了什么呢？”
“我去外面吃饭的时候遇见个女孩，她借我的手机打电话给她的母亲们。”
“然后女孩的母亲们为了答谢我，居然给了我300镑！”
“哇，好幸运！！”
“你看看吧。”
她将那几张钞票拿了出来。
“没有去兑换吗？”
“没有，外币感觉也挺稀奇的，留着也不错…”
“嗯嗯～说得是呢，这样也不错。”
“话说…你会看钻石吗？”
“钻石？”
她似乎很惊讶。
“很久前一个人送的…虽然有证书，但我对此一窍不通…”
“别人送的？”
“嗯，那次的相遇，也很奇妙…”
“那次，她还拿巧克力给我吃…就是你之前买的那口味。”
“…”
没有什么好怀疑的了。
“我在那之后…一直想要再见到她…”
“但是…一直以来…没有…”
…
突然间，我吻上她柔软的嘴唇。
然后，分开。
“那个…？”
她的神情惊讶，脸变得红了起来。
“或许，那人就是我呢？”
我对她如此说。
…
…
“哈…啊啊！！唔——————“
在我的舔舐下，她的身体不住地颤抖，口中发出喘息声。
真可爱。
“月经…来了…”她如此说。
诶？
她来月经了吗？
我向下看。
她的一些经血流到被子上，形成了血痕。
部分经血还沾在了阴部毛发和阴唇上。
我舔舐她流下的经血。
咸腥的味道，不错。
我舔着不断流下来的经血。
“唔…呜…”她发出声音。
好可爱，好可爱。
她的经血和体液一直流出来。
我则一直在舔舐着。
我的舌与她的阴部触碰的水声不绝于耳。
我的下体也感到温润而湿热。
“呀————啊…哈啊….”
她的抖动，她的喘息。
我又一次，让如此可爱的她高潮。
“哈——啊—哈…啊…”
她的身体，又在一阵一阵地颤抖。
“那个…恒晖…也让我来吧…？想要你也高潮…”
…
…
一夜，二人高潮不断。
疯狂地渴望着彼此。
爱着彼此。
继续吧————
家人。
…
…
室内酒席，烟雾环绕。
地上，掉着未燃尽的烟头。
“来吧！嫂子，我敬你一杯！”
嘭——碰杯的声音响起。
“谢谢哥。”
“儿子死了你也不要太伤心哈，嫂子，你还要活呢！”
“嗯，行，谢谢哥。”
“你为啥还要供你女儿？我老早就觉得她真恶，果然，最蠹人心。”
“她不听俺嘞话。”
“大嫂，你真是心软，我听到她对你儿子做的事就感觉背后一寒，这人咋能这么变态啊？”
“*，嫂嫂的女儿做了什么呀？”席上，一个年幼的男孩问。
“你这么小嘞小孩就别知道这些了，怕你晚上睡不着觉。”
“什么呀？”
“过来过来。”另一个年幼的男孩敲敲那个男孩。
“怎么了？”
“我也很怕…但你真要听吗？”
“嗯。”
“这也是我听说的啊…好像就是，那个女人把她弟弟打得头破血流…还把鸡…鸡用烟烫和剪掉了，听说大人看见的时候是剪了一半但还连着一半…哎呀，说说我觉得都好恐怖…”
“呀————？？这么…这么恐怖…喵！！我也会被女人这样对待吗？”
“说不定啊…我们都要小心一点女人…不然把我们剪了怎么办…那得痛死了….”
“呀——！不要…不要…说了…女人好可怕…女人会把我鸡鸡剪掉…不要…”
男孩对于下体被剪有着极端的恐惧。
“如果被女人剪掉鸡鸡的话…我就不是男生了…不要…”
“没事的啦，你不会被剪的啦，只要不接触女人。”
“真的？”
“来嘛，我来教你怎么样当男人。”
“艾？”
两小孩趁大人聊天的时候溜了出去。
…
…
“好痛好痛————！！！菊花痛痛痛！！！”
“闭嘴，这是做男人必经的事！！爽得很！！老子试过！”
男孩打了另一个男孩一拳。
“不要…好痛…你在霸凌我…我要找妈妈…”
“女人会把你鸡鸡剪掉的，你听我说没？”
“但你这样做也是不对的…好痛…好痛…”
“不听老子话？我揍死你！”
男孩又打了几拳。
“不要打我，呜呜呜…”
“你这样反抗，老子更兴奋了！”
“不要…不要——！！”
“别给我出声！”
男孩用手捂住另一个男孩的嘴。
“呜————呜呜——呜！！”
男孩面目潮红，惚惚闪闪的大眼睛流下眼泪，打在男孩雪白透红的玉肌上。
男孩的无毛小白美腿正在不停地抖动着。
男孩的身体正在被玷污。
男孩再也不纯洁干净了。
嘛，真的纯洁干净过吗？
是男孩勾引人吧？那萌萌的脸，小白美腿和雪白透红的美丽玉肌就是证明，男孩不长成这样勾引人不就好了？
明明很爽吧？
是在欲拒还迎吧？
得了便宜还卖乖。
脏男孩。
…
“啊…呜呜…”
男孩没有力气，只得栽在地上，不停地颤抖哭泣。
男孩弱小无助，只得被人操弄。
如此痛苦的模样真是美丽动人。
“废物一个，两下就烂了？”
“啊…啊…妈妈…救我…”
“这里没你妈！你个废物！”
男孩朝地上的男孩使劲踢了一脚。
“喵呜————！！！”
男孩发出了猫叫声，模样很是蠢萌，让人有捏碎的冲动。
人之常情。
“妈妈！！妈妈————无论谁，救我…”
男孩似乎力气要耗尽一般。
“吵死了！！！”
站着的男孩拿起一块石头，砸在了躺着的男孩脸上。
“唔————“
男孩身体除脸外没被压的部分突然往上挣扎一抬————
然后，再无波动。
